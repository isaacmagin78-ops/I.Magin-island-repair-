# briefs/

Drop video briefs here — one Markdown file per video, based on
[`../VIDEO-BRIEF-TEMPLATE.md`](../VIDEO-BRIEF-TEMPLATE.md).

Workflow:

1. Upload/copy media into `public/assets/` (images, videos, music, voiceover, logos subfolders).
2. Write the brief here, e.g. `briefs/2026-07-tyson-vertical.md`.
3. Ask Claude Code: *"Render the video described in briefs/2026-07-tyson-vertical.md."*
4. The rendered MP4 lands in `out/`.
